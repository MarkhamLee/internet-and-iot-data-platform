## Documentation 

This folder contains documentation on the platform's architecture, CICD & GitOps, features, types of workloads, etc. While much of the Documentation lives in this folder, in some cases this detailed documentation for a workload type (e.g., Agentic Workloads) lives in the same folder as the workloads themselves, and in those cases this Readme document will just link to that information.
